# Security Policy

## Supported Versions

This project is currently in MVP stage and does not maintain versioned security backports.
Security fixes are applied to the default branch.

## Reporting A Vulnerability

For now, report security issues via GitHub Issues:

- [Open a new issue](https://github.com/forPelevin/hlcut/issues/new)

When filing, include:

- Impact and threat model (what can be abused and by whom)
- Reproduction steps
- Minimal proof of concept
- Suggested mitigation (if known)

If the issue includes secrets, do not paste raw keys/tokens in the report.
